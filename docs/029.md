# 目标

你的类的一个实例如果没有对象就没有类的用处。如果我们有类，那么我们需要创建类的对象来使用类解决我们的问题。您可以使用 new 关键字创建您的类的对象。$ obj class = new my class()；

```
<?php
class MyClass
{
    function createObject()
    {
        echo "creating object of class MyClass"; 
    }
}

$bar = new MyClass;
$bar->createObject();
?>
```
