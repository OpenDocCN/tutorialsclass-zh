### 什么是 CSS？

*   CSS 代表“级联样式表”。
*   级联样式表(CSS)是一种用于指定网页设计和格式的语言。换句话说，CSS 用于控制 web 文档的样式。
*   CSS 使用 HTML 标签。

学习 CSS 是如此有趣。您可以应用不同的 CSS 属性来更改网页格式。

下面是一个 CSS 代码的例子:

```
body {
    background-color: blue;
}

p {
    font-family: "Times New Roman";
    font-size: 22px;
}
```

**细节示例**
在这里，你可以看到像 body 和 p 这样的 Html 标签，然后我们对这些标签应用了不同的 CSS 格式。
