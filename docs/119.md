# WordPress 开发

在 WordPress 开发教程中，我们将介绍一些先进的 WordPress 概念。我们将学习 WordPress 代码、API 和内置函数。学习 WordPress 主题开发或者 WordPress 插件开发，要有 PHP，HTML，CSS，JavaScript 的基础知识。
